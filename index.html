Coding Projects

HttpStatusCodeChecker is a utility I whipped up one night to check the HTTP status code of all the resources of a given URL, this utility uses Jsoup for DOM parsing and the java.net package for request and response code handling. You can download the jar with all dependencies packed here. For more information on the utility and how to use it , read the enclosed readme file.

Chess is an object oriented implementation of the board game I wrote for my high school computer science course, using the java.swing toolkit and JPanels. Different chess pieces and their legal moves logic are in separate classes, all of which inherit from a Piece superclass. The other game logic (such as checking for checkmates) are contained in GamePanel, which is instantiated by MainApplet, a wrapper class that enables the program to be embedded in a browser. The source code for the program can be found here.
